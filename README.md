# Scrabble Coding Challenge - Hasbro

## Dictionnary

For english dictionnaty, I've decide to use the official [Collins Scrabble Words (2019)](https://en.wikipedia.org/wiki/Collins_Scrabble_Words), as it's the official word authority for tournament Scrabble in the USA and Canada.

## Score calculation

I've tried to get the proper scoring of Scrabble. So I base my scoring system on the [English letter distribution](https://en.wikipedia.org/wiki/Scrabble_letter_distributions) from Wikipedia.
